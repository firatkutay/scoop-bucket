# scoop-bucket

Scoop bucket for [`comrade`](https://github.com/firatkutay/cli-comrade) ‚Äî
a cross-platform AI CLI companion for the terminal (TR/EN).

Published automatically by `cli-comrade`'s `goreleaser` release pipeline
(`scoops` block in `.goreleaser.yaml`). Do not edit `comrade.json` by hand ‚Äî
every release overwrites it.

## Install

```powershell
scoop bucket add firatkutay https://github.com/firatkutay/scoop-bucket
scoop install comrade
```
